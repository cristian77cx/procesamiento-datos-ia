# 🚀 DEMO INTERACTIVA - Instrucciones

## ¿Cómo probar la aplicación?

### Opción 1: Ejecutar localmente (Recomendado)

1. **Clona el repositorio:**
```bash
git clone https://github.com/cristian77cx/procesamiento-datos-ia.git
cd procesamiento-datos-ia
```

2. **Instala las dependencias:**
```bash
pip install -r requirements.txt
```

3. **Ejecuta la aplicación:**
```bash
streamlit run app.py
```

4. **Abre tu navegador en:**
```
http://localhost:8501
```

## ✨ ¿Qué puedes hacer en la demo?

### 📁 **Pestaña 1: Cargar Datos**
- **Sube tu propio archivo CSV o Excel** (arrastra y suelta)
- **O genera datos de ejemplo** con un clic
- Ve estadísticas instantáneas: filas, columnas, tamaño, valores nulos

### 🧹 **Pestaña 2: Limpieza**
- **Limpieza automática** con un botón
- Elimina duplicados, valores nulos y outliers
- Ve métricas en tiempo real del antes y después
- **Descarga los datos limpios** en CSV

### 📊 **Pestaña 3: Análisis**
- **Estadísticas descriptivas** completas
- **Visualizaciones interactivas:**
  - Distribuciones de variables
  - Matriz de correlación
  - Boxplots para detectar outliers
  - Gráficos de variables categóricas
- Selecciona qué columnas visualizar

### 🤖 **Pestaña 4: ML Prep**
- **Preparación automática para Machine Learning**
- Codificación de variables categóricas
- Escalado de features (Standard, MinMax, Robust)
- División automática en Train/Test (80/20)
- **Descarga datasets listos** para entrenar modelos

## 🎯 Casos de uso reales

### Ejemplo 1: Analista de Marketing
**Problema:** Tienes datos de clientes con información incompleta y duplicada.

**Solución:**
1. Sube tu archivo `clientes.csv`
2. Limpia automáticamente (elimina duplicados y nulos)
3. Analiza patrones en el comportamiento de compra
4. Descarga datos limpios para tu CRM

### Ejemplo 2: Científico de Datos
**Problema:** Necesitas preparar un dataset para entrenar un modelo de ML.

**Solución:**
1. Carga tu dataset
2. Limpia y analiza los datos
3. Prepara features con escalado automático
4. Descarga Train/Test sets listos para usar

### Ejemplo 3: Estudiante/Investigador
**Problema:** Tienes datos de una encuesta y necesitas visualizarlos.

**Solución:**
1. Genera datos de ejemplo o sube los tuyos
2. Explora estadísticas descriptivas
3. Crea visualizaciones profesionales
4. Exporta resultados
