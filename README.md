# 🚀 AI Data Processing Platform

**Desarrollado por:** Cristian Pineda  
**Rol:** Analista de Datos Junior

## 📊 Descripción

Aplicación web interactiva para procesamiento, limpieza y preparación de datos para modelos de Inteligencia Artificial.

## ✨ Características

- 📁 **Carga de datos:** Sube archivos CSV o Excel
- 🎲 **Datasets de ejemplo:** Ventas, Clientes, Empleados
- 🧹 **Limpieza automática:** Elimina duplicados, nulos y outliers
- 📊 **Análisis exploratorio:** Estadísticas y visualizaciones
- 🤖 **Preparación ML:** Codificación, escalado y división de datos
- ⬇️ **Exportación:** Descarga datos procesados

## 🛠️ Tecnologías

- Python
- Streamlit
- Pandas & NumPy
- Matplotlib & Seaborn & Plotly
- Scikit-learn

## 🚀 Uso

1. Selecciona un dataset de ejemplo o sube tu propio archivo
2. Configura las opciones de limpieza en el sidebar
3. Explora los análisis y visualizaciones
4. Descarga los datos procesados
